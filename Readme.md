# The-Catch-the-Spies-challenge

Набор утилит для обработки и анализа данных в рамках задачи обнаружения аномалий и подозрительных активностей.

Ключенвые этапы:
1. Парсинг: чтение и извлечение информации из файлов в форматах CSV, JSON, PDF, YALM, EXEL
2. Очистка и фильтрация данных
3. Агрегация: объединение данных в единую структуру
4. Выделение определенных групп людей по признакам
